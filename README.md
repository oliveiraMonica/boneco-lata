# boneco-lata
Boneco de lata feito com C++ e OpenGL

## Camera
- Teclas A e D = câmera gira 360°
-  tecla W boneco anda
- Teclas 1 à 6 câmeras em diferentes angulos.

## Boneco:
- glutSolidCylinder (tecla l)
- glutWireSphere (tecla b)
